排序特性​

  1. 稳定：直接插入，冒泡交换

  2. 不稳定：希尔插入，快速交换，简单选择，堆选择

O(n2)O(n^2)O(n2)

  1. 空间复杂度快排式log。 归并排序是ON。 基数排序是。 Ord。 其他都是。 O1。

  2. 交换类的排序排序趟数和原始序列有关。基本有序时，冒泡交换有利，快速交换不利。

  3. 简单选择和折半插入的关键字比较次数和原始序列无关。

  4. 与初始排列序列无关的是基数排序。

  5. 冒泡交换、快速交换、简单选择，堆选择可以使经过一趟排序能够保证一个关键词到达最终位置。
