## char 和 int 的转换

```cpp
int x = 5;

char i = x; //越界
char i = x - '0'; //越界
char i = x + '0'; //'5'

char y = '0';

int j = y; //48 ascii
int j = y - '0'; //0
int j = y + '0'; //96
```
