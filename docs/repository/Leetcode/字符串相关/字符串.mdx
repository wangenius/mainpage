# search

## 58-最后一个单词的长度

```cpp title="字符串中最后一个字符的长度"
int lengthOfLastWord(string s) {
    int pos = s.size() - 1;
    while (s[pos] == ' ') --pos;
    int len = 0;
    while (pos >= 0 && s[pos] != ' ')
    {
        ++len;
        --pos;
    }
    return len;
}
```
