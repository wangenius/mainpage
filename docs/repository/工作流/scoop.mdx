## 安装

### 环境需求

1. PowerShell 5 +
2. NET Framework 4.5 +

### 指定安装位置

指定安装位置（配置环境变量）：

```shell title="以管理员身份打开PowerShell"
$env:SCOOP='D:\Applications\Scoop'
[Environment]::SetEnvironmentVariable('SCOOP', $env:SCOOP, 'User')
$env:SCOOP_GLOBAL='F:\GlobalScoopApps'
[Environment]::SetEnvironmentVariable('SCOOP_GLOBAL', $env:SCOOP_GLOBAL, 'Machine')
```

### CLI安装

首先确保你能访问 `raw.githubusercontent.com`

```shell showLineNumbers
`Invoke-Expression (New-Object System.Net.WebClient).DownloadString('https://get.scoop.sh')
iwr -useb get.scoop.sh | iex`
```

## Scoop 使用

:::info Scoop官方维护的仓库
- [main](https://github.com/ScoopInstaller/Main) - 默认仓库
- [extras](https://github.com/lukesampson/scoop-extras) - 默认仓库的补充超级强大
可以直接通过`scoop bucket add <repo_name>`安装

镜像仓库：

- [main](https://codechina.csdn.net/mirrors/ScoopInstaller/Main.git)
- [extras](https://codechina.csdn.net/mirrors/lukesampson/scoop-extras.git)
- 更多的可以在[CSDN GitHub加速计划上](https://codechina.csdn.net/mirrors)找到
:::



### Scoop仓库换为国内仓库

#### 安装git

>（如果你的电脑有git可以跳过）

```bash title="安装git"
scoop install git
```

#### 删除原有仓库

```bash
scoop bucket list
scoop bucket rm main
```

#### 添加镜像仓库

```bash
scoop bucket add main https://codechina.csdn.net/mirrors/ScoopInstaller/Main.git
scoop bucket add extras https://codechina.csdn.net/mirrors/lukesampson/scoop-extras.git
```
#### 更新一下

```bash
scoop update
```

### 使用Scoop安装常用软件

```bash
scoop install 7zip aria2 python vscode typora vlc sumatrapdf
```

最后，愉快的使用scoop吧

### 安装指定版本的软件

可以在软件名字后面指定完整的版本号

```bash
scoop install python@3.7.8
```
