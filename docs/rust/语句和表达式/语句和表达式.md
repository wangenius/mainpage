语句会执行一些操作但是不会返回一个值，需要分号结尾；

表达式会进行求值，然后返回一个值，不能有分号；

调用一个函数，调用宏，调用 `{}` 语句块，都是一个表达式，会有返回值

```Rust
let x = x + 1; // 语句
let y = y + 5; // 语句
x + y // 表达式
```
