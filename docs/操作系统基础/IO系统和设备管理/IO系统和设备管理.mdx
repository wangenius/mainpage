IO系统由IO软件和IO硬件两部分

## 设备

1. 使用特性：存储设备、人机交互设备、网络通讯设备

2. 信息交换：字符设备、块设备

3. 传输速率：低速、中速、高速

4. 共享属性：独占、共享、虚拟

5. 是否阻塞：阻塞IO、非阻塞IO
