# 设备驱动程序

设备驱动程序：常以进程的形式存在。IO进程与设备控制器之间的通信程序。其主要任务是接收上层软件发来的抽象IO要求，如`read`或`write`命令，转换为具体要求后发送给设备控制器，启动设备去执行；此外，它也将从设备控制器发来的信号传送给上层软件。**每一类设备配置一种驱动程序，也可为非常类似的两类设备配置一个驱动程序。**
